## Start the game
You'll be in command mode at first.
`. adv`

## Walk around
`h, j, k, l`

## Insert mode
After typing out a command, press enter to do stuff

### Enter insert mode
`i`

### Actions:
  - look key -- key is looked up in a dictionary. Try "look me"
  - go direction

### Switch back to command mode
`q` don't forget to press enter.
